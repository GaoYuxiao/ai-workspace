# Skill市场 产品交互文档

## 1. 导航设计

### 1.1 主导航
#### 主导航设计
**定义**：位于页面顶部的全局导航栏，用于切换平台的主要功能模块。

**设计要求**：
- **位置布局** - 位于页面最顶部，深色背景
- **菜单结构** - 包含：聊天器、智能体开发（带红点提示）、空间管理、服务市场（带红点提示）、数据编辑管理、CodeReview、平台管理
- **视觉样式** - 深色导航栏，白色文字，激活状态显示蓝色高亮
- **交互状态** - 支持默认、悬停、激活状态，部分菜单支持红点提示

### 1.2 左侧导航
#### 左侧导航设计
**定义**：页面左侧的二级导航栏，用于切换不同的市场类型。

**设计要求**：
- **位置布局** - 位于页面左侧，固定宽度约200px
- **菜单结构** - 包含：模型市场、MCP市场、Skill市场（当前选中）
- **视觉样式** - 深色背景，选中项显示蓝色高亮背景
- **交互状态** - 支持默认、悬停、激活状态

### 1.3 分类导航
#### 分类导航设计
**定义**：左侧导航下方的标签筛选区域，用于按标签筛选Skill。

**设计要求**：
- **位置布局** - 位于左侧导航栏下方，包含搜索框和分类列表
- **菜单结构** - 包含：搜索框、全部（107）、云概念、APIGW（96）、文档（1）、编程（1）、支持（1）、上云数据集（5）、腾讯云（1）等
- **视觉样式** - 列表形式，显示分类名称和数量，选中项显示蓝色背景
- **交互状态** - 支持点击切换分类，显示悬停效果

## 2. 页面结构

### 2.1 左侧分类筛选区
**功能定位**：提供MCP的分类筛选和搜索功能，帮助用户快速定位目标MCP。

**组件构成**：
- **搜索输入框** - 位于分类列表顶部，placeholder"搜索指定关键字"，支持实时搜索
- **全部分类** - 显示"全部"及总数量（如107），蓝色图标
- **分类列表** - 垂直列表展示各分类及对应数量
  - 云概念
  - APIGW（96）
  - 文档（1）
  - 编程（1）
  - 支持（1）
  - 上云数据集（5）
  - 腾讯云（1）
  - 其他分类...

### 2.2 Skill卡片展示区
**功能定位**：以卡片网格形式展示Skill列表，每个卡片展示Skill的核心信息。

#### 2.2.1 顶部操作栏
**组件构成**：
- **搜索框** - 右侧主搜索框，placeholder"搜索中文关键词、描述、更新人"，支持全文搜索
- **排序按钮** - "排序" 按钮
- **排序下拉菜单** - "创建时间（降序）" 等排序选项

#### 2.2.2 Skill卡片展示区
**功能定位**：以卡片网格形式（3列布局）展示Skill列表，每个卡片展示Skill的核心信息。

**卡片布局**：
- 网格布局：每行3列
- 卡片背景：白色（#FFFFFF）
- 卡片边框：1px solid #E5E6EB
- 卡片圆角：8px
- 卡片阴影：悬停时显示阴影效果

**数据字段**：
- **Skill图标** - 位于卡片左上角，圆角方形，尺寸48px × 48px
  - 橙色图标：文档处理类（pdf, pptx, xlsx, docx等）
  - 蓝色图标：开发工具类（skill-creator, mcp-builder等）
  - 紫色图标：代码审查类（code-review-excellence, canvas-design等）
- **Skill名称** - 图标右侧，加粗显示，字体16px，字重600
- **作者用户名** - Skill名称下方，灰色文字，字体14px
- **Skill描述** - 位于卡片中部，多行文本展示（约3-4行）
  - 支持文本截断，超出部分显示省略号
  - 字体14px，行高1.5
  - 颜色：#646A73
- **下载量** - 位于卡片底部左侧
  - 下载图标 + "共 xxx 次下载"
  - 字体12px，颜色：#8F959E

### 2.3 Skill详情侧滑区
**功能定位**：点击Skill卡片后从右侧滑出详情面板，展示Skill的完整信息和使用说明。

**面板结构**：
- **面板头部** - 显示Skill基本信息和操作按钮
- **面板主体** - 显示Skill详细描述和使用文档
- **面板关闭** - 右上角关闭按钮（X图标）

**头部信息字段**：
- **Skill图标** - 左侧显示Skill的标志性图标，尺寸48px × 48px，圆角方形
- **Skill名称** - 图标右侧，显示完整名称，字体20px，字重600，颜色#1F2329
- **创建人** - 名称右侧，灰色文字，格式"创建人: [用户名]"
- **共享范围** - 创建人右侧，显示图标和文字，如"共享范围: 🏢 公共 🔍"
- **下载量** - 共享范围右侧，显示下载图标和统计，格式"共 xxx 次下载"
- **下载按钮** - 右上角主按钮，黑色背景（#000000），白色文字"下载"，带下载图标
- **关闭按钮** - 右上角X图标按钮，灰色

**主体内容结构**：
- **Skill简介** 
  - 位于头部下方
  - 段落形式的功能描述和使用场景说明
  - 字体14px，行高1.6，颜色#1F2329
  
- **使用方法**
  - 标题：字体16px，字重600，颜色#1F2329
  - 内容：分步骤说明（1. 2. 3...）
  - 支持有序列表展示
  - 支持代码块展示命令示例（灰色背景）
  - 支持缩进层级显示

**面板样式规范**：
- 面板宽度：建议最大宽度1200px
- 面板背景：白色（#FFFFFF）
- 面板内边距：32px
- 头部高度：72px
- 头部边框底部：1px solid #E5E6EB
- 内容区段落间距：24px
- 标题上间距：32px
- 滑出动画时长：300ms
- 背景遮罩：半透明黑色，透明度0.5

## 3. 交互流程设计

### 3.1 浏览Skill流程
**流程描述**：用户浏览和筛选Skill的完整流程。

**关键步骤**：
1. 用户进入Skill市场页面
2. 默认按下载量排序展示Skill列表
3. 可切换到按发布时间排序
4. 页面滚动加载更多Skill卡片

### 3.2 搜索Skill流程
**流程描述**：用户通过搜索查找特定Skill的流程。

**关键步骤**：
1. 用户在搜索框输入关键词
2. 系统实时过滤匹配的Skill
3. 展示搜索结果列表
4. 用户点击目标Skill卡片查看详情

### 3.3 查看Skill详情流程
**流程描述**：用户查看Skill详细信息的流程。

**关键步骤**：
1. 用户点击Skill卡片
2. 从右侧滑出详情面板（动画时长300ms）
3. 查看Skill头部信息（名称、作者、共享范围、下载量）
4. 滚动查看详细内容（简介、使用方法、预期输出）
5. 点击"下载"按钮下载Skill
6. 点击右上角X按钮或点击遮罩层关闭面板

### 3.4 上传Skill流程
**流程描述**：用户上传自己创建的Skill的流程。

**关键步骤**：
1. 用户点击页面右上角"上传 Skill"按钮
2. 弹出"上传 Skill"对话框（居中显示）
3. 填写必填信息：
   - **名称**：在输入框中输入Skill名称（必填，红色星号标识）
   - **上传Skill包**：点击上传区域或拖拽ZIP文件到上传区域
     - 支持.zip格式
     - 文件大小限制：最大100MB
     - 上传区域显示上传图标和提示文字
   - **使用说明**：在多行文本框中详细描述Skill的功能、使用方法和注意事项（必填，红色星号标识）
     - 建议包含：功能介绍、使用步骤、系统要求等
     - 支持多行文本输入
4. 设置共享范围（可选）：
   - 默认状态："当前仅您可以访问"（开关关闭）
   - 开启共享：切换开关至打开状态，设置为"共享"
5. 检查填写内容：
   - 系统验证必填字段是否完整
   - 验证文件格式和大小是否符合要求
6. 点击"确认上传"按钮提交：
   - 上传成功后关闭对话框，显示上传成功提示
   - Skill进入个人Skill列表
   - 如设置为共享，Skill将出现在市场列表中供其他用户查看和下载
7. 取消操作：点击"取消"按钮或右上角X关闭对话框，放弃上传
