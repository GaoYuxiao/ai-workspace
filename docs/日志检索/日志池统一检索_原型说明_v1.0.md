## 1. 页面导航结构

### 1.1 信息架构

```
日志平台
├── 📊 日志检索 (默认首页)
│   ├── 日志池选择器

│   └── 结果展示区

│   ├── 日志池详情/设置
│   │   ├── 基本信息
│   │   ├── 采集项管理
│   │   ├── 元数据管理
│   │   ├── 别名配置
│   │   └── 分词设置

│
├── 📋 条件组管理
│   ├── 条件组列表
│   ├── 条件组详情/编辑
```
