# 项目目录说明

根目录只保留项目运行必需文件，各模块内容按以下方式存放。

| 目录 | 说明 |
|------|------|
| **docs/** | 所有产品/技术文档 |
| docs/日志检索 | 客户端日志检索、统一检索、日志池、日志聚类等 PRD、方案、原型说明 |
| docs/mcp | MCP 工具列表、告警 MCP 文档、evals 评估结果 |
| docs/测试 | 测试流程图、bklog 选择器映射等 |
| docs/grok | Grok 内置规则、功能说明、demo |
| **reports/** | 分析报告与数据产出 |
| reports/多别名设置验证 | 多别名相关的问题排查报告、分析数据、HTML 报告 |
| reports/分析报告 | 各类业务/技术分析（bkmonitor、bkrepo、数据库、Trace 等） |
| **scripts/** | 可执行脚本 |
| scripts/多别名分析 | 多别名分析、报告生成、排查脚本 |
| scripts/图表 | 图表生成脚本及 chart_data 等 |
| scripts/其他 | 其他工具脚本（如飞书、行情等） |
| **rum-app-management/** | RUM 应用接入模块（PRD、前端、截图） |
| **skill/** | Cursor Agent Skills 源码及打包的 .skill 文件 |
| **testcase/** | 自动化测试用例、测试报告 |
| **archive/** | 个人或临时文件归档 |

查找文件时优先到对应模块目录下查找。
